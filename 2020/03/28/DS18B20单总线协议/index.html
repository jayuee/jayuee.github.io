单总线协议和18B20的使用

一、单总线协议介绍

单总线协议（1-wire）采用单根信号线，即可传输时钟，又能传输数据，而且数据传输是双向的。

二、时序介绍

1.初始化时序

  1. 总线复位：（高电压）

  2. 拉低总线：大于480us小于960us

  3. 拉高总线：15~60us

  4. 接收返回：（0则成功，1则失败）

2.写0/1时序

1. 拉低总线
2. 给总线赋值（0或1）
3. 延时60us
4. 如果写两位数据，写入的间隙要拉高总线大于1us

tips:

1、15us之内必须把所需数据送到总线上

2、18b20,15~60us则对总线采样

3、若低电平，写入的位是 0

​ 若高点平，写入的位是 1

先写低位

3.读0/1时序

1. 拉低总线1us
2. 拉高总线
3. 延时大于5us小于15us
4. 读取总线电平（高：1，低：0）
5. 在延时60us内完成一个指令

读两位数据间要把总线拉高1us（与写一样）

经过8次循环，原来最高位变成了最低为。即：先读的是最低位
